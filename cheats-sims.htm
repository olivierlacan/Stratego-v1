Les Sims

Activez les codes en appuyant sur + + + C en cours de jeu. Tapez ensuite les codes suivants :

- klapaucius - 1000$

- set_hour # - Change l'heure (# de 1 à 24)

- set_speed # - Change la vitesse du jeu (# de -1000 à +1000)

- water_tool - Rajouter de l'eau

- map_edit on/off - Éditer la carte

- grow_grass # - Faire pousser la végétation (# de 1 à 150)

- interests - Voir les intérêts de vos sims

- autonomy # - Changer ce que pensent les sims

- edit_char - Mode création de personnages

- quit - Quitte la partie

- ; - Sépare deux codes

- ! - Répète le dernier code
